## 8. What Happens During Training

1. **Warmup (20 eps)**: Robot stays still (VLA ref=0, residual=0). Fills buffer.
2. **SAC starts (ep 21+)**: Random joint deltas → Jacobian → Cartesian motion.
   Robot explores workspace. Initial motions are large/random.
3. **Learning (ep 50+)**: SAC discovers which joint deltas lead to reward.
   Actions become smaller, more directed toward insertion.
4. **Convergence (ep 200+)**: SAC reliably guides peg into hole.
   Residuals are small, precise corrections.

## 9. Future Improvements

1. **Working VLA from HPC**: Swap checkpoint → VLA provides meaningful reference
2. **servoJ mode**: Direct joint control (skip Jacobian) for faster loop
3. **2-camera OpenPI config**: Remove wrist_image_right duplication
4. **RLPD demos**: Load SERL demo data into replay buffer (50% demo sampling)
5. **β annealing**: Reduce BC weight as Q-values improve
